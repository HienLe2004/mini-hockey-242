# Player's movement:

## Move: WASD / Arrow keys
## Dash: F / Numpad 0
## Grouping: G / Numpad 1
## Change player: H / Numpad 2

# Change scene:
## From main menu: 
N -> Gameplay scene
O -> Gameplay with AI scene
## From gameplay/gameplay with AI scene:
N -> Game over scene
M -> Main menu scene
## From game over scene:
N -> Gameplay scene
O -> Gameplay with AI scene
M -> Main menu scene
